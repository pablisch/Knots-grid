Some letters with diacritical marks: à á â ã ò ó ô õ

They are displayed by writing constructs such as a&#768; to dsiplay à.


Reserved entities are also written using enitity names or enitity numbers, e.g. & is coded in HTML as &#38, while < is coded as &#60.


The € symbol can be coded using an entity name, &euro;, an entity number, &#8364; or as an entity hexadecimal, &#x20AC;.


You can find many of the symbol codes here, such as ☠ and ☣.


Emojis are simply characters that cannot be typed on a normal keyboard and so they each have an entity number just like any other character.

Any letter, symbol or emoji coded with an entity number should start with &# followed by the number and end with ; so that the browser understands that it should display the designated character, symbol, emoji, etc.

Here are some favourites: &#128516; will give you 😄, &#128525; will give you 😍 and &#128151; will give you 💗.
